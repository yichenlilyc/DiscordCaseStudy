Introduction

Background:

Discord is an online communication platform which initially served game players, but gradually developed to satisfy the requirements of various groups of people. Discord has been favored by gamers because of its simplicity: Unlike other online communication tools, it does not include too much workload to affect the performance of computers and thus negatively impact the game experience.

 

Problem Statement:

Through over year upgrades of the website, Discord installed multiple functions to cover demand of a wider range of people. However, it also makes the interface more complicated and confuses users who are not familiar with the platform. While the interface supports more functions, the design becomes more important. Our group believes Discord can improve on some details of UX design of the interface, like a better categorization of windows, to achieve a better user experience.

 

Design Process:



Discover

Qualitative Research:

We interviewed 10 users to find out what they think of discord’s interface and how they feel about using it. The main questions we asked include:

    • What do you mainly use Discord for?
    • How frequently do you use discord? Everyday/ A few times a week/ A few times a month/ Hardly ever/ Never use
    • What background theme do you use in Discord?
    • How easy is it to find the functions and sections that you need in Discord? 0-5
    • What do you think Discord needs to be improved?
    • How would you rate Discord's interface design? 0-5
    • How would you rate the overall feeling of using Discord？0-5

What we can improve:

  • Find game friends:
    • Allow users to organize servers in a better way
    • Give users easier access to the server management tab
    • Give users more intuitive way to manage notifications so they will not miss important messages and will not get distracted by non-essential messages
    • Allow users to open members tab in a separate window
  • Chat with friends:
    • Allow users to organize messages in a more organized way
    • Allow users to customize the styles of the fonts of the message
    • Enable threadings in the message channel to make it easier for users to follow the chats
    • Make the direct message tab more visible and accessible
  • Play
    • Allow users to view and manage system usages by discord

 


Ideate

Potential Solutions:

    • Conducted a discussion session to find potential solutions
    • Most of the issues and outcome goals are personal (not general to every user)
      • It is our hard to design a general solution for every user
    • Customization is mentioned many times, especially for the layouts and aesthetics
    • Multiple specific solutions proposed, including delivering a “lite” version, adding in new dedicated notification setting board, etc.

 

First Solution:

 

Optimization:

  • Modularized Customization System with Plugins and Marketplaces
    • For users who does not want to create their own plugins:
      • They can go to the marketplace to install plugins
    • For users who would like to create their plugins:
      • They can use provided libraries and apis to create their plugins, similar to visual studio code
